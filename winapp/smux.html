smux

Windows向け GPU加速ターミナルマルチプレクサー

DirectX 12 によるGPU加速描画、ペイン分割、WASMプラグイン、組み込みブラウザ、tmux互換CLI、セッション永続化。 Windows Fluent Design（Mica Alt / Acrylic）に対応したモダンなターミナル環境を提供。

v0.8.4
Rust + Zig + gpui
開発ツール
AGPL-3.0
GitHub 設定 & コマンド一覧 一覧に戻る

主な機能

⚡

GPU加速描画

DirectX 12 / wgpu によるテキストレンダリング。gpuiフレームワークで滑らかな60fps描画を実現。

⬜

ペイン分割

水平（Ctrl+D）・垂直（Ctrl+Shift+D）分割。ドラッグでリサイズ可能なボーダー付き。

📋

ワークスペース

タブ形式のワークスペース管理。Ctrl+1〜9でクイックジャンプ、リネーム、ドラッグ並び替え対応。

🖥️

tmux互換CLI

tmuxコマンドのドロップイン代替。Named Pipes IPCでGUIとCLIをシームレスに連携。

🧩

WASMプラグイン

wasmtimeランタイムでWebAssemblyプラグインを実行。安全に機能を拡張可能。

🌐

組み込みブラウザ

WebView2ベースのブラウザパネル。CDP自動化サポートでブラウザ操作も可能。

💾

セッション永続化

ワークスペース、分割、作業ディレクトリを保存・復元。再起動後も作業状態を維持。

🎨

Fluent Design

Windows 11のMica Alt / Acrylicバックドロップに対応。OSに溶け込むモダンなUI。

🔍

コマンドパレット

Ctrl+Shift+Pでファジーコマンド検索。すべての操作にキーボードだけでアクセス。

📊

サイドバー

Gitブランチ、作業ディレクトリ、リスニングポート、通知バッジを表示。

🔄

自動アップデート

起動時にGitHub Releasesをチェック。ワンクリックで最新版に更新。

👨‍💻

シェルプロファイル

PowerShell Core、Windows PowerShell、CMD、WSL、Git Bashを自動検出。
