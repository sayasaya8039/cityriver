Screenshot Translator

Translate and summarize text from screenshots

v1.0.0
Chrome / Edge
Manifest V3
GitHub 一覧に戻る

スクリーンショットをAIが即座に翻訳・要約するChrome拡張機能

---

概要

「スクショ翻訳」は、画面上の外国語テキストをスクリーンショットで撮影し、AIが自動で翻訳・要約するChrome拡張機能です。OpenAI GPT-4oまたはGoogle Gemini 2.0 Flashを使用して、高精度な翻訳を実現します。

主な機能

キャプチャ方法

  • 範囲選択キャプチャ - 画面の必要な部分だけをドラッグで選択
  • タブ全体キャプチャ - 表示中のページ全体を一発でキャプチャ
  • クリップボード貼り付け - 他アプリでコピーした画像をそのまま翻訳

その他の機能

  • 翻訳履歴 - 過去の翻訳結果を自動保存（最大50件）
  • カスタムショートカット - キーボードショートカットで素早くアクセス
  • ダークモード対応 - システム設定に連動

インストール

Chrome Web Storeから（推奨）

1. Chrome Web Storeの拡張機能ページにアクセス

2. 「Chromeに追加」をクリック

開発版をインストール

1. このリポジトリをクローンまたはダウンロード

2. Chromeで chrome://extensions を開く

3. 右上の「デベロッパーモード」を有効化

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードしたフォルダを選択

初期設定

1. 拡張機能をインストールすると設定画面が開きます

2. 使用するAIプロバイダーを選択（OpenAI または Google Gemini）

3. 選択したプロバイダーのAPIキーを入力

4. 「保存」をクリック

APIキーの取得方法

OpenAI

1. OpenAI Platform にアクセス

2. アカウントを作成またはログイン

3. 「Create new secret key」でAPIキーを生成

Google Gemini

1. Google AI Studio にアクセス

2. Googleアカウントでログイン

3. 「Get API key」でAPIキーを生成

使い方

基本的な使い方

1. 翻訳したいページを表示

2. 拡張機能のアイコンをクリック

3. キャプチャ方法を選択：

  • 「範囲を選択してキャプチャ」- 翻訳したい部分をドラッグで選択
  • 「タブ全体をキャプチャ」- ページ全体をキャプチャ
  • 「クリップボードから貼り付け」- コピー済みの画像を使用

4. AIが自動で翻訳・要約を表示

キーボードショートカット

| 機能 | Windows / Linux | Mac |

|------|-----------------|-----|

| 範囲選択キャプチャ | Ctrl + Shift + S | Cmd + Shift + S |

| タブ全体キャプチャ | Ctrl + Shift + T | Cmd + Shift + T |

| クリップボード貼り付け | Ctrl + Shift + V | Cmd + Shift + V |

ショートカットは chrome://extensions/shortcuts でカスタマイズ可能です。

対応AI

| プロバイダー | モデル | 特徴 |

|-------------|--------|------|

| OpenAI | GPT-4o | 高精度、多言語対応 |

| Google Gemini | Gemini 2.0 Flash | 高速、コスト効率 |

プライバシー

  • 画像データはAI APIに送信されますが、端末やサーバーには保存されません
  • APIキーはブラウザのローカルストレージに保存され、外部に送信されません
  • 翻訳履歴はブラウザのローカルストレージにのみ保存されます

技術仕様

  • Manifest Version: 3
  • 対応ブラウザ: Chrome 99以上
  • 必要な権限:
  • activeTab - アクティブタブのキャプチャ
  • storage - 設定・履歴の保存
  • clipboardRead - クリップボードからの画像読み取り
  • scripting - 範囲選択UIの表示
  • tabs - タブ情報の取得
