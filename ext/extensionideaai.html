ExtensionIdeaAI - 拡張機能アイデアジェネレーター

ブラウジング中のページを分析して、AIが拡張機能のアイデアを提案します。オフラインで動作します。

v1.0.8
Chrome / Edge
Manifest V3
一覧に戻る

ブラウジング中のページを分析して、AIがChrome拡張機能のアイデアを提案してくれるオフライン対応のChrome拡張機能です。

個人開発者が、毎日のウェブ閲覧からインスピレーションを得て、革新的な拡張機能を開発できるように設計されています。

🔥 特徴

ページ自動分析

  • ページのDOM構造、テキスト、機能を自動的に分析
  • ページタイプ（ECサイト、SNS、ニュースサイト等）を自動判定
  • 改善の機会（アクセシビリティ、パフォーマンス等）を検出

AIブレインストーム

  • Chrome Built-in AI API（Gemini Nano）でアイデアを生成
  • オフライン完全対応 - ネットが切れても動作
  • ページの特性に応じた最適なアイデアを提案

豊富なアイデア情報

各アイデアには以下の情報が含まれます：

  • 魅力的なタイトルと機能説明
  • 必要な権限（permissions, host_permissions）
  • 実装のヒント（content script, popup等）
  • ターゲットユーザー層
  • バズりやすさ/需要の予測

カスタムプロンプト

  • デフォルト：バランスの良いアイデア
  • プライバシー重視：データ保護・プライバシー強化
  • 生産性向上：作業効率改善
  • アクセシビリティ：障碍者向け支援

保存・管理機能

  • 気に入ったアイデアをローカルストレージに保存
  • サイドパネルで保存済みアイデアを一覧表示
  • 検索・フィルタリング機能
  • JSON形式でエクスポート可能

🚀 インストール方法

Chromeウェブストア（予定）

近日公開予定

手動インストール（開発版）

1. このリポジトリをクローンまたはダウンロード

2. Chromeを開き、chrome://extensions にアクセス

3. 「デベロッパーモード」をオン

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ExtensionIdeaAI フォルダを選択

📖 使い方

基本的な使い方

1. 任意のウェブページを開く

2. ツールバーの 💡 ExtensionIdeaAI アイコンをクリック

3. ページが自動で分析されます

4. 「アイデアを生成」ボタンをクリック

5. AIが拡張機能のアイデアを3〜5個提案します

ショートカット

  • Ctrl+Shift+E（Mac: Cmd+Shift+E）: ポップアップを開く
  • Ctrl+Shift+K（Mac: Cmd+Shift+K）: サイドパネルを開く

アイデアの保存

1. 提案されたアイデアの 💾 ボタンをクリック

2. サイドパネルで保存済みアイデアを管理可能

3. エクスポート機能でバックアップ作成

🛠️ 技術スタック

  • Manifest V3: 最新の拡張機能規格
  • Chrome Built-in AI API: Gemini Nano（オフライン動作）
  • Vanilla JavaScript: 外部ライブラリなし
  • Local Storage: アイデアのローカル保存

📁 ファイル構成


        ExtensionIdeaAI/
        ├── manifest.json          # 拡張機能の設定
        ├── background.js          # Service Worker（AI連携）
        ├── content.js             # Content Script（ページ分析）
        ├── popup.html/css/js      # ポップアップUI
        ├── sidepanel.html/js      # サイドパネルUI
        ├── icons/                 # アイコン（16x16, 32x32, 48x48, 128x128）
        └── create_icons.py        # アイコン生成スクリプト

🔧 開発

開発環境

  • Chrome 127+（Built-in AI API対応バージョン）
  • Python 3.14+（アイコン生成用）

プロジェクトのセットアップ


        # リポジトリのクローン
        git clone https://github.com/yourusername/ExtensionIdeaAI.git
        cd ExtensionIdeaAI
        
        # アイコン生成（既に生成済み）
        python create_icons.py
        
        # Chromeに読み込んで開発

Built-in AI APIについて

この拡張機能はChromeの実験的な機能であるBuilt-in AI APIを使用しています。

有効にする方法：

1. chrome://flags にアクセス

2. 「Prompt API for Gemini Nano」を有効化

3. Chromeを再起動
