AI Chat Exporter

AI会話をMarkdown/HTML/Text/PDFでエクスポート（Gemini, Claude, ChatGPT, Grok, Perplexity対応）

v3.0.0
Chrome / Edge
Manifest V3
一覧に戻る

Geminiの会話履歴をMarkdown形式でエクスポートするChrome拡張機能です。

機能

  • Markdown形式: 会話を編集可能なMarkdown形式でエクスポート
  • 表の変換に対応
  • コードブロックの改行を保持
  • HTMLコンテンツを適切にMarkdownに変換

インストール方法（デベロッパーモード）

1. Chromeブラウザを開き、アドレスバーに chrome://extensions/ と入力してEnterキーを押します

2. 右上の「デベロッパーモード」をONにします

3. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします

4. このフォルダ（gemini-exporter-extension）を選択します

5. 拡張機能が読み込まれ、ツールバーにアイコンが表示されます

使い方

1. Geminiにアクセスし、会話を開始します

2. エクスポートしたい会話が表示されている状態で、ツールバーの拡張機能アイコンをクリックします

3. ポップアップが表示されるので、「Markdownエクスポート」ボタンをクリックします

4. Markdownファイルが自動的にダウンロードされます

  • ファイル名は最初のユーザーメッセージ（最大30文字）またはGeminiのタイトルから自動生成されます

技術スタック

  • Manifest V3: 最新のChrome拡張機能仕様
  • Content Scripts: Geminiページからデータを抽出
  • Turndown: HTMLからMarkdownへの変換ライブラリ

ファイル構成


        gemini-exporter-extension/
        ├── manifest.json          # 拡張機能の設定ファイル
        ├── background.js          # バックグラウンドスクリプト
        ├── content.js             # コンテンツスクリプト（データ抽出）
        ├── popup.html             # ポップアップUI
        ├── popup.css              # ポップアップのスタイル
        ├── popup.js               # ポップアップの制御スクリプト
        ├── exporter.js            # Markdownエクスポート機能の実装
        ├── images/                # アイコン画像
        │   ├── icon16.png
        │   ├── icon32.png
        │   ├── icon48.png
        │   └── icon128.png
        └── lib/                   # 外部ライブラリ
            └── turndown.min.js    # HTMLからMarkdownへの変換

特徴

  • 軽量: Markdownエクスポート専用のシンプルな設計
  • 表の対応: HTMLテーブルをMarkdown形式に正確に変換
  • コードブロック: 改行を保持したコードブロックの出力
  • スマートなファイル名: 最初のユーザーメッセージまたはGeminiのタイトルから自動生成

注意事項

  • この拡張機能は gemini.google.com でのみ動作します
  • 会話の構造によっては、一部のデータが正しく抽出されない場合があります
  • 大量のメッセージを含む会話の場合、エクスポートに時間がかかる場合があります
