ニュース記事バイアスチェッカー

ニュース記事の政治的バイアスを分析し、視覚的に表示します。

v1.3.0
Chrome / Edge
Manifest V3
GitHub 一覧に戻る

ニュース記事の政治的バイアス（左派/中立/右派）をAIが分析し、視覚的に表示するChrome拡張機能です。

機能

  • ニュース記事を自動で抽出・分析
  • 左派/中立/右派のバイアス度をパーセンテージで表示
  • カラフルなバーグラフで視覚的に確認
  • ダークモード/ライトモード自動対応
  • 複数のAI APIに対応（OpenAI, Gemini, Claude）
  • ページ移動・タブ切り替え時に自動更新

スクリーンショット

（準備中）

インストール方法

1. このリポジトリをダウンロードまたはクローン

2. Chromeで chrome://extensions を開く

3. 右上の「デベロッパーモード」を有効化

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. ダウンロードしたフォルダを選択

設定方法

1. 拡張機能アイコンを右クリック →「オプション」

2. 使用するAPIを選択（OpenAI / Gemini / Claude）

3. 対応するAPIキーを入力

4. 「設定を保存」をクリック

APIキーの取得

| API | 取得先 |

|-----|--------|

| OpenAI | OpenAI API Keys |

| Gemini | Google AI Studio |

| Claude | Anthropic Console |

使い方

1. ニュース記事のページを開く

2. 自動で記事が分析される

3. 拡張機能アイコンをクリックして結果を確認

4. 必要に応じて「再分析」ボタンで再度分析

バイアスの判定基準

左派・リベラル傾向

  • 憲法9条改正に反対、護憲的立場
  • 政権・与党に批判的
  • 社会福祉・再分配を重視
  • ジェンダー平等・LGBT権利を重視

右派・保守傾向

  • 憲法改正に積極的
  • 政権・与党を支持・擁護
  • 防衛力強化・日米同盟重視
  • 伝統的家族観を重視

中立

  • 複数の立場を公平に紹介
  • 事実報道に徹し、意見を控える
  • 賛否両論を併記

対応ニュースサイト

  • 毎日新聞
  • 朝日新聞
  • 読売新聞
  • 日経新聞
  • NHK
  • Yahoo!ニュース
  • その他一般的なニュースサイト

技術スタック

  • Chrome Extension Manifest V3
  • Vanilla JavaScript
  • CSS Variables（ダークモード対応）
  • AI APIs
  • OpenAI GPT-4o-mini
  • Google Gemini 2.0 Flash
  • Anthropic Claude 3.5 Haiku

プライバシー

  • APIキーはローカルストレージにのみ保存（同期なし）
  • 記事の内容は分析のためにAI APIに送信されます
  • 分析結果はローカルにのみ保存されます
