デジタル疲労を軽減するマインドフルネス・ブラウザアシスタント Chrome拡張機能

概要

Mindfulness Browser Assistant は、長時間のブラウジングによるデジタル疲労を軽減するための Chrome 拡張機能です。Opera の Air 機能にインスパイアされ、呼吸ガイド・瞑想タイマー・バイノーラルビート・ストレッチガイドなど、包括的なウェルネス機能をブラウザ上で提供します。

グラスモーフィズムデザインによる美しいサイドパネルUIで、作業を中断せずにマインドフルネスを実践できます。

機能一覧

🌬 呼吸ガイド

アニメーション付きの呼吸サークルで、3つのパターンから選択可能：

  • リラックス (4-7-8) — 副交感神経を活性化する深い呼吸
  • 集中 (4-4-4-4) — バランスの取れた集中力向上パターン
  • 安らぎ (2-0-4) — すぐにできるクイックリリーフ

セッション時間: 1分 / 2分 / 3分 / 5分

🧘 瞑想タイマー

ベルサウンド付きの瞑想セッション（528Hzソルフェジオ周波数）：

  • クイック・カーム — 3分間の短い休息
  • ボディスキャン — 5分間の全身リラックス
  • ディープ・フォーカス — 10分間の深い集中
  • エクステンド・ピース — 15分間の本格瞑想

🎵 オーディオミキサー

Web Audio APIによるリアルタイム音響合成：

バイノーラルビート（4プリセット）

| プリセット | ベース周波数 | ビート周波数 | 脳波帯域 |

|-----------|------------|------------|---------|

| 集中 | 200Hz | 10Hz | Alpha |

| リラックス | 150Hz | 6Hz | Theta |

| 睡眠 | 100Hz | 4Hz | Theta |

| 活力 | 250Hz | 14Hz | Beta |

環境音（5トラック同時再生可能）

  • 🌧️ 雨 — ホワイトノイズ + バンドパスフィルタ + LFO変調
  • 🌲 森 — ピンクノイズ + 3声の鳥のさえずり（周波数変調）
  • 🔥 焚き火 — ブラウンノイズ + クラックリング変調
  • 🌊 波 — ピンクノイズ + 8秒周期のウェーブリズム
  • 💨 風 — ピンクノイズ + スイープフィルタ + ガスト変調

各トラックの個別ボリューム調整、マスターボリューム、プリセット保存/読み込みに対応。

🤸 ストレッチガイド

ステップバイステップのタイマー付きストレッチルーティン：

  • 首・肩 — 4ステップ / 約2分
  • 全身リセット — 6ステップ / 約3分

⚙ 設定

  • テーマ: ライト / ダーク / 自動
  • グラス不透明度: グラスモーフィズムの透明度調整
  • 通知: 間隔（15〜60分）、強度（やさしい / 普通 / 強制）
  • オーディオ: マスターボリューム、バイノーラル有効/無効
  • センシング: ドゥームスクロール検知

📊 セッション統計

  • 今日の集中時間・休憩回数・気分スコア
  • 過去7日間の集中時間バーチャート
  • 週間サマリー（合計集中・休憩・平均気分・記録日数）
  • 5段階の気分ログ（😔😕😐🙂😊）

🛡 ドゥームスクロール検知

  • スクロール速度と持続時間をリングバッファで監視
  • 5分以上の高速スクロール（500px/秒超）を検知
  • 自動で休憩を促すトースト通知

🔔 休憩リマインダー

3段階の通知強度：

  • やさしい — サイドパネル内トースト
  • 普通 — Chrome通知
  • 強制 — フルスクリーンオーバーレイ（呼吸/ストレッチへのショートカット付き）

アーキテクチャ


        Popup (ポップアップUI)
          │
          ├─ chrome.scripting.executeScript
          │    └─ CustomEvent('__mindfulness_cmd')
          │
        Content Script (Shadow DOM内React App)
          ├─ document.addEventListener('__mindfulness_cmd')
          ├─ サイドパネルUI (380px 右サイドバー)
          ├─ Web Audio API (バイノーラル + 環境音)
          └─ ScrollMonitor (ドゥームスクロール検知)
          │
        Background Service Worker
          ├─ chrome.alarms (休憩リマインダー)
          ├─ chrome.notifications (システム通知)
          └─ chrome.storage (設定管理)

セットアップ

必要環境

  • Node.js 18+
  • npm / pnpm

インストール


        git clone https://github.com/sayasaya8039/Mindfulness_Browser_Assistant.git
        cd Mindfulness_Browser_Assistant
        npm install

ビルド


        npm run build

dist/ フォルダにビルド成果物が生成されます。

Chrome に読み込み

1. Chrome で chrome://extensions を開く

2. 右上の「デベロッパーモード」を有効にする

3. 「パッケージ化されていない拡張機能を読み込む」をクリック

4. dist フォルダを選択

開発モード


        npm run dev

ファイル変更を監視し、自動リビルドします。

使い方

1. 任意のウェブページで、画面右端の ‹ トグルボタンをクリック

2. または拡張機能アイコンからポップアップを開き、「マインドフルネスパネルを開く」 をクリック

3. タブで機能を切り替え（呼吸 / 瞑想 / 音響 / ストレッチ / 設定）

4. パネル外をクリックすると閉じます
