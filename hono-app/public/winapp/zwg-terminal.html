ZWG Terminal が選ばれる理由

他のターミナルとは一線を画す、こだわりの機能群をご覧ください。

⚡

GPU レンダリング

GPUI を活用したハードウェアアクセラレーテッドな描画で、どんな負荷でも滑らかに動作します。遅延ゼロのタイピング体験を。

🦎

Zig で書かれた堅牢なコア

Zig 言語による型安全で高性能なコアロジック。メモリ安全性と実行速度を両立した、信頼性の高いアーキテクチャです。

🌐

WASM プラグインシステム

WebAssembly ベースのプラグイン機構により、任意の言語で拡張機能を開発可能。サンドボックス環境で安全に動作します。

🚀

超高速レンダリング 600fps 維持

GPUI による GPU 直接描画で、常時 600fps を維持。大量のログ出力やリアルタイム処理でもフレームドロップなし。従来のターミナルとは次元の違う滑らかさを体感できます。

🎨

美しい UI / テーマ

精緻にデザインされたデフォルトテーマと、細かくカスタマイズできる設定システム。フォント・カラー・レイアウトを自由に。

🪟

Windows ネイティブ最適化

WSL2 との完全統合、Windows Terminal プロトコルのフルサポート。Windows 開発者のために最適化されています。

// tech_specs

技術仕様

ZWG Terminal を支える技術スタックの詳細です。

  • 言語 Zig 0.13+
  • レンダリングエンジン GPUI (GPU Accelerated)
  • プラグインランタイム WebAssembly (WASM)
  • 対応 OS Windows 10/11 (x64)
  • WSL2 対応 ✓ フル統合
  • 起動時間 < 20ms
  • ライセンス MIT
  • 現バージョン v0.1.0-beta
# ZWG Terminal 設定例 (zwg.toml)

[general]
  theme = "dark-cyber"
  font_family = "JetBrains Mono"
  font_size = 14

[rendering]
  gpu_acceleration = true
  fps_limit = 120

[wasm]
  plugins_dir = "~/.zwg/plugins"
  sandbox = true
