リファレンス

smuxv0.8.4

設定項目、キーボードショートカット、CLIコマンドの完全リファレンス。

⚙️ 設定ファイル

設定は以下の優先順位で読み込まれます。

1
最優先（smux設定）%APPDATA%\smux\config.json
2
フォールバック（Ghostty互換）%APPDATA%\ghostty\config
3
デフォルトCatppuccin Mocha テーマ

🎨 カラー設定

colors
プロパティ型デフォルト説明
colors.backgroundu320x1e1e2e背景色
colors.foregroundu320xcdd6f4前景色（テキスト）
colors.cursoru320xf5e0dcカーソル色
colors.selection_bgu320x585b70選択範囲の背景色
colors.selection_fgu320xcdd6f4選択範囲の前景色
colors.ansi[0-15][u32; 16]CatppuccinANSI 16色パレット

🔤 フォント設定

font
プロパティ型デフォルト説明
font.familyString"Cascadia Code"フォントファミリー
font.sizef3214.0フォントサイズ（pt）
font.line_heightf321.3行の高さ（倍率）
font.ligaturesbooltrueリガチャ有効

⚙️ 一般設定

config.json
プロパティ型デフォルト説明
sidebar_visiblebooltrueサイドバー表示
sidebar_widthf32240.0サイドバー幅（px）
shellString"powershell.exe"デフォルトシェル
scrollback_linesusize10000スクロールバック行数
confirm_close_processbooltrueプロセス実行中の閉じる確認
workspace_placementenum"End"新規WS配置 (Top / AfterCurrent / End)
browser_host_modeenum"popup"ブラウザモード (child / popup)

🖼️ 背景画像

background
プロパティ型デフォルト説明
background.image_pathString?null背景画像の絶対パス
background.image_opacityf320.3不透明度（0.0~1.0）

🔧 カスタムキーバインド設定

keybindings でアクション名にキーストロークを割り当て。null で無効化。

// config.json 内
"keybindings": {
  "Quit": "ctrl-shift-q",
  "NewTab": null // 無効化
}

💻 CLI — ペイン操作

$ smux split-window -h
右にペイン分割
$ smux split-window -v
下にペイン分割
$ smux list-panes
全ペイン一覧
$ smux select-pane -t %0
ID指定でフォーカス
$ smux focus-pane left
方向指定でフォーカス移動
$ smux kill-pane -t %0
ペインを閉じる
$ smux capture-pane -t %0 -p
ペイン内容をキャプチャ
$ smux send-keys -t %0 "ls" Enter
キー入力を送信

💻 CLI — ワークスペース

$ smux workspace-new [name]
新規作成
$ smux workspace-list
一覧
$ smux workspace-switch 2
切替
$ smux workspace-rename 0 "Dev"
リネーム
$ smux workspace-close
閉じる
$ smux next-workspace
次へ
$ smux prev-workspace
前へ
$ smux workspace-pin 3
ピン切替

💻 CLI — タブ操作

$ smux add-tab -t %0
新規タブ追加
$ smux close-tab -t %0
タブを閉じる
$ smux cycle-tab -t %0
次のタブ
$ smux select-tab -t %0 2
タブ選択

💻 CLI — ブラウザ

$ smux browser-toggle
表示/非表示
$ smux browser-close
閉じる
$ smux browser-navigate <url>
URLへナビゲート
$ smux browser-eval "js"
JS実行
$ smux browser-snapshot
スナップショット
$ smux browser-click <sel>
クリック
$ smux browser-fill <sel> "text"
テキスト入力
$ smux browser-screenshot
スクリーンショット

💻 CLI — セッション

$ smux session-save
セッション保存
$ smux session-restore
セッション復元
$ smux has-session
起動確認
$ smux info
ステータス（JSON）

💻 CLI — 通知

$ smux notify "Title" "Body"
通知送信
$ smux notification-list
通知一覧
$ smux notification-read <id>
既読にする
$ smux notification-clear
全クリア

💻 CLI — 設定管理

$ smux config-get
全設定をJSON出力
$ smux config-get font.size
値を取得
$ smux config-set font.size 16
値を変更
$ smux config-reload
再読み込み

💻 CLI — tmux互換

Claude Code teammate-mode用。未実装サブコマンドもexit 0で応答。

$ smux new-session
新規セッション
$ smux new-window
新規ウィンドウ
$ smux list-windows
ウィンドウ一覧
$ smux show-options
オプション表示
$ smux display-message -p
メッセージ出力

📝 設定例

%APPDATA%\smux\config.json
{
  "colors": {
    "background": "0x1e1e2e",
    "foreground": "0xcdd6f4",
    "cursor": "0xf5e0dc",
    "selection_bg": "0x585b70",
    "selection_fg": "0xcdd6f4"
  },
  "font": {
    "family": "Cascadia Code",
    "size": 14.0,
    "line_height": 1.3,
    "ligatures": true
  },
  "sidebar_visible": true,
  "sidebar_width": 240.0,
  "shell": "powershell.exe",
  "scrollback_lines": 10000,
  "confirm_close_process": true,
  "workspace_placement": "End",
  "browser_host_mode": "popup",
  "background": {
    "image_path": null,
    "image_opacity": 0.3
  },
  "keybindings": {
    "Quit": "ctrl-shift-q"
  }
}
